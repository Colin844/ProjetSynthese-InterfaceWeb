Les Fonctions Fléchées

Les fonctions fléchées (arrow functions) sont une syntaxe concise pour définir des fonctions en JavaScript. Elles permettent d'éviter l'utilisation explicite du mot-clé function et ne possèdent pas leur propre contexte this.

Manipulation des Tableaux

Trois méthodes essentielles permettent de manipuler les tableaux :

  • map() : Permet de transformer chaque élément d'un tableau en retournant un nouveau tableau.
  • filter() : Filtre les éléments d'un tableau en fonction d'une condition donnée.
  • reduce() : Réduit un tableau à une seule valeur en appliquant une fonction sur chaque élément.

Destructuring

Le destructuring, ou affectation par décomposition, permet d'extraire des données d'objets ou de tableaux en une seule ligne de code :

const [a, b] = [1, 2]; // a = 1, b = 2

Spread Operator et Rest Parameters

Le spread operator (...) permet d'étaler les éléments d'un tableau ou d'un objet, tandis que les rest parameters permettent de collecter plusieurs éléments en un seul tableau :

function sum(...numbers) {
    return numbers.reduce((acc, n) => acc + n, 0);
}

Méthodologie BEM

La méthodologie BEM (Block, Element, Modifier) est une convention de nommage utilisée en CSS pour structurer et organiser le code de manière cohérente. Exemple :

.block__element--modifier
